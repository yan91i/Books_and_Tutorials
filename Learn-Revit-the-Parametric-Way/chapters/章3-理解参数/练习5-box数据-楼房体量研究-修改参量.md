# 练习5: box数据-楼房体量研究-修改参量

## 练习目的

在设计中根据概念设计的需要做出各种不同数据类型的办公楼的族库。

## 打开练习文件 

双击打开下载资源包中“章3-理解参数\练习5-box数据-楼房体量研究-修改参量”文件夹中的"模型块Box-办公楼的体量-Begin.rfa"和"办公楼-数据.xlsx"，打开如下方所示的练习文件。

![box数据-楼房体量研究-修改参量01.png](/images/box数据-楼房体量研究-修改参量/box数据-楼房体量研究-修改参量01.png)

这个练习文件中有一个蓝色的六面体，它被命名为办公楼。通常意义上，办公楼的形态可以从多种角度进行分类。从剖切面的形式上划分，办公楼可以被分成方形的，长字形的，L形的，圆形的，椭圆形的等。但是在城市设计和建筑设计的概念方案阶段，为了粗略统计面积，并且大体上研究建筑物的布局，从剖切面的形式上，办公楼被分成方形的和一字形的两种类型。

![box数据-楼房体量研究-修改参量02.png](/images/box数据-楼房体量研究-修改参量/box数据-楼房体量研究-修改参量02.png)

这个Excel文件中有着策划需要的，办公楼的长度，宽度和高度的数据。具体的数据如下：

- 一共有24种不同数据类型的办公楼。
- 从长度和宽度上划分，办公楼有36 * 36，43 * 43，45 * 27，58 * 26，70 * 27，93 * 25（单位：米 * 米），共6种类型。
- 从高度上划分，办公楼有76，94，115，和144（单位：米），共4种高度。

强调一下：这24种不同数据类型的办公楼不太可能会在一个真实的建筑成果里全部用上，但是可以用它们来做概念设计布局的摆布和粗略的工程量的统计。它们可以解决下面的问题：

- 如果面积被限定，概念设计有多少种布局方式；
- 如果建筑的限高被确定，概念设计有多少种布局方式；
- 如果建筑的平面形式被确定，概念设计有多少种布局方式。

> 注明：**族的命名解释如下**
> 
- 例如：办公_方_36*36_115。其中
- **办公**：办公楼
- **方**：方形的
- 36*36：楼的长度是36米，宽度是36米
- **115**：楼的高度是115米
